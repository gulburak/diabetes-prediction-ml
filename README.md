# Diabetes Prediction — Machine Learning Project

## Overview
This project focuses on predicting diabetes using medical and demographic patient data. A machine learning classification pipeline was developed using Python and scikit-learn.

## Dataset
- Training set: 73,718 samples × 10 features
- Test set: 26,146 samples × 9 features

## Methods
- Data cleaning and preprocessing
- Feature encoding and scaling
- Random Forest classification
- Hyperparameter tuning with RandomizedSearchCV
- Evaluation using classification metrics

## Results
- Validation F1-score: 0.81

## Tools & Libraries
Python, pandas, NumPy, scikit-learn, matplotlib
